# LineageOS 14 Device tree for Xiaomi Redmi Note 2 (HERMES)

This branch is device tree for building LineageOS 14.1 Rom for Xiaomi Redmi Note 2 (HERMES).

  - Mic
  - Wi-Fi
  - Bluetooth
  - Internet

  - Hotspot
  - Camera rear/front
  - SD Card
  - IR Blaster
  - Online/Offline charging
